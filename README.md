# Onivo

Onivo es una aplicación backend desarrollada con .NET siguiendo los principios de Clean Architecture.

El objetivo del proyecto es mantener una separación clara entre la lógica de negocio, los casos de uso, la infraestructura y la capa de presentación, facilitando el mantenimiento, las pruebas y la escalabilidad del sistema.
